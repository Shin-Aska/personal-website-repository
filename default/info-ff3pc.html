Final Fantasy 3 (PC Port)

four orphans on their adventure destinied by the crystals of light to save the world once again

Overview

Final Fantasy 3 is a remake of the NES game of the same name. It is a port of a the IOS/Android version which is a port of the original Nintendo DS remake. The 3D remake gives improve graphics than the original NES. Other than that, it also adds backstories from the four orphans which the original game did not have. The 3D remake also added additional content as well to add more replayability factor for old-time FF players. So if this is your first time playing Final Fantasy, then I highly suggest you play this game over the NES version first!

Gameplay

The RPG game is basically a turn-based style where each characters get to execute one move per each turn. In battle, these characters have their own speed which dictates the priority on who gets to do the move first. The battle system itself is very simple to pick-on so you need no to worry about technicalities of the system.

The storyplaying remains true to the original. Unlike modern RPGS that contains a log. This one doesn't, just like the original. The game isn't too cryptic as you can play the game in a linear fashion and can even complete the game without resorting to a playthrough or walkthrough guide.

Compatibility

Since the game itself contains graphics that do not contain any sophisticated graphical effects, you can play the game in pretty much any computer that was made on 2010. Most of the screenshots I have where made on a very low-end laptop (notebook they call it) which you can check the specs here (click on this link) and my TravelMate is an earlier revision, which contains 2GB of RAM instead. Hard drive space isnt an issue either because the game consumes less than 1GB in file size. So more or less, we can say that the game itself can be ran on a potato computer.
