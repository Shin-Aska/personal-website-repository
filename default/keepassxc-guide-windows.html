KeepassXC Guide [Windows]

Downloading

KeepassXC can be downloaded here[1]. Once clicking on this page, you will be greeted by a series of options which one to download. There is no correct version to choose from as each of them has their own purpose. I will explain these options further:

  1. Official Packages: Installer 64 bit - This is for Windows 10 and 11 64bit users
  2. Official Packages: Portable 64 bit - This is for Windows 10 and 11 64bit users that wants to store the app itself on a USB storage or other portable solution
  3. Official Packages: Legacy 64 bit - This is for older Windows versions such as 8 and 8.1
  4. Official Packages: Legacy 32 bit - Use this when using a 32bit Windows regardless of version
  5. 3rd-Party Distributions: Chocolatey - Use this if you are using Chocolatey and want to enjoy automatic updates
  6. 3rd-Party Distributions: Winget - Use this if you are using Winget and want to enjoy automatic updates
For most users, installer 64bit is the common option. This guide will also be using the installer 64bit as well. Once you are done downloading, run the installer (skip this option for portable users) then run the application (For those the use the installer, you can find the application on the Start menu; Whereas the portable one is on the decompressed zip file)

If in any case you encounter any problems with the Installer and Portable 64bit, please download and run this installer[2].

Getting Started

Now that you have finished installing KeepassXC and you are already running the application as instructed earlier, now you must need to learn a couple of things about this password manager.

Before you are able to begin, you must first create a KeepassXC password/database file. This file will contain all your passwords including TOTP and other metadata however rest assured that it will be encrypted.
